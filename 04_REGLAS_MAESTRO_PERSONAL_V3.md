# 04 - Reglas de Maestro Personal

Archivo: maestro_personal.xlsx. Columnas: ID Operador, Nombre, Area, Rol, Lider, Activo, Observacion. Áreas válidas: Bodega, Despachos, Mixto, Excluir.

Cruce: ID Operador normalizado exacto; luego nombre normalizado; luego tokens; luego fuzzy sin ambigüedad.

Inactivo o Excluir no cuenta. No encontrado = PERSONAL_NO_MAESTRO. En mensual/rango continuar y reportar.

Roles para metas PPT Bodega actuales: Montacarguita/Montacarguista, Pick and drop/Pick and Drop, Trilateral.
